# Extract patient name from a character string

Extract patient name from a character string

## Usage

``` r
extract_name(input)
```

## Arguments

- input:

  A character string containing the patient name after the labno

## Value

The patient name as a string

## Examples

``` r
extract_name("WS123456_12345678_AnnaKarenina")
#> [1] "AnnaKarenina"
```
